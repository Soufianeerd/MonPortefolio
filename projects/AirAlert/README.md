# AirAlert PWA 🏃‍♂️⚡️

**AirAlert** est une Progressive Web App sportive ultra-premium conçue pour le suivi des performances, des objectifs et de la progression athlétique.

## 🚀 Stack Technique
- **Backend** : Python Flask
- **Base de données** : SQLite (SQLAlchemy)
- **Frontend** : HTML5, CSS3, JavaScript Vanilla
- **PWA** : Service Workers, Web Manifest (Installable, Offline-ready)

## 🎨 Direction Artistique
Le design suit les codes du "Quiet Luxury" et de l'esthétique sportive moderne (Apple Fitness+, Nike Training Club, Strava Premium).
- Palette : Crème, Beige, Bleu Électrique, Bleu Nuit.
- Typographie : Plus Jakarta Sans.
- Composants : Mockup iPhone CSS, Glassmorphism, Animations de révélation.

## 📂 Structure du projet
```
airalert-pwa/
│── app.py             # Point d'entrée Flask
│── database.py        # Configuration SQLite
│── requirements.txt   # Dépendances
│── static/            # Assets statiques
│── templates/         # Pages HTML (Jinja2)
│── data/              # Base de données
```

## 🛠 Installation
1. Installez les dépendances : `pip install -r requirements.txt`
2. Lancez l'application : `python app.py`
3. Accédez à `http://localhost:5001`
