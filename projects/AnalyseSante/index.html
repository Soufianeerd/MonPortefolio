À quoi ça sert ?
Analyse décisionnelle : Ce dashboard permet d'analyser la posture et la dynamique de mouvement d'un sujet. Il aide les praticiens à détecter des anomalies de stabilité ou des pics d'intensité lors d'exercices spécifiques.

Architecture : cette page est la version frontend pure (HTML/CSS/JS, simulateur intégré). Le projet existe aussi en version full-stack dans le dossier backend/ — serveur Flask (Python) + daemon C++ qui ingère un jeu de données réel issu de capteurs inertiels.
